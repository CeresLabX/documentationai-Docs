## Overview

CeresLab provides a robust set of features to help you create, organize, and collaborate on your documentation wiki. You can build comprehensive knowledge bases with intuitive editing tools, structured organization, powerful search, version control, and media handling. These core capabilities make it easy to maintain high-quality, searchable content.

## Page Creation and Editing Workflows

Follow these steps to create and edit pages efficiently.

<Steps>
  <Step title="Create a New Page" icon="plus">
    Navigate to the search bar and enter a new page title. Click `<kbd>Create</kbd>` to start editing.

    ```

    bash
    Search for "New Page Title" → Click "Create"
    ```
  </Step>
  <Step title="Edit Content" icon="edit">
    Use the visual editor or wiki markup. Add headings, lists, and links.

    <CodeGroup tabs="Visual,Wiki Markup">
    ````html
    <h2>Main Section</h2>
    <ul>
      <li>Item 1</li>
    </ul>
    ````
    ````wiki
    == Main Section ==
    * Item 1
    ````
    </CodeGroup>
  </Step>
  <Step title="Preview and Save" icon="save">
    Preview changes, then click `<kbd>Publish</kbd>` to save. Add a summary for the edit.
  </Step>
</Steps>

<Callout kind="tip">
  Enable rich text editing in your preferences for a WYSIWYG experience.
</Callout>

## Organizing Content with Namespaces and Categories

Structure your wiki logically using namespaces and categories.

| Feature       | Description                                                                 | Example Usage                  |
|---------------|-----------------------------------------------------------------------------|--------------------------------|
| Namespaces    | Prefix pages like `Help:`, `Category:`, or `Perfumery:` to group content.   | `Perfumery:Materials`          |
| Categories    | Tag pages with `[[Category:Name]]` for dynamic indexing.                     | `[[Category:Perfumery]]`       |

Add categories at the bottom of pages:

````wiki
[[Category:Documentation]]
[[Category:Features]]
````

<Tabs>
  <Tab title="Namespaces" icon="layers">
    Namespaces separate content types. Create custom ones like `Project:` for your docs.
  </Tab>
  <Tab title="Categories" icon="tag">
    Categories create browsable indexes. View all at `Special:Categories`.
  </Tab>
</Tabs>

## Search and Discovery Functionalities

CeresLab's search helps you locate content instantly.

<Expandable title="Advanced Search Options" default-open="true">
  Use filters for namespaces, categories, and full-text search. Example query: `perfumery insource:Main`.
</Expandable>

Key features include:
- Full-text search across all pages
- Namespace-specific results
- Recent changes and watchlists for monitoring

## Collaboration and Version History

Collaborate seamlessly with built-in tools.

<Steps>
  <Step title="Watch Pages" icon="eye">
    Click `<kbd>Watch</kbd>` to get notifications on changes.
  </Step>
  <Step title="View History" icon="clock">
    Access page history to compare revisions and revert changes.
  </Step>
  <Step title="Discuss" icon="message-circle">
    Use talk pages for feedback: `Talk:PageName`.
  </Step>
</Steps>

## Media and File Management

Upload and manage images, documents, and more.

1. Go to `Special:Upload`.
2. Select files and add descriptions.
3. Embed with `[[File:filename.ext|thumb|Caption]]`.

<Callout kind="info">
  Supported formats: PNG, JPG, SVG, PDF. Max size: 10MB per file.
</Callout>
